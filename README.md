# CoreAI Scan

### Intelligent Dataset Quality Analysis & ML Readiness Engine

CoreAI Scan is an AI-powered dataset intelligence platform that automatically analyzes uploaded datasets and provides data quality assessment, machine learning readiness evaluation, algorithm recommendations, outlier detection, correlation analysis, and AI-driven cleaning suggestions.

The platform helps data scientists, machine learning engineers, analysts, and students quickly understand dataset quality before model development.

---

## Features

### Dataset Health Analysis

* Dataset Health Score
* Dataset Readiness Score
* Dataset Quality Ranking
* Dataset Grade (A+ to D)
* Risk Level Assessment

### Data Quality Inspection

* Missing Value Detection
* Duplicate Row Detection
* Dataset Completeness Calculation
* Numerical and Categorical Feature Analysis
* Outlier Detection using IQR Method

### Machine Learning Readiness

* Regression vs Classification Dataset Identification
* ML Readiness Evaluation
* AI Verdict Generation
* Feature Quality Assessment

### AI Recommendations

* Intelligent Cleaning Suggestions
* Missing Value Handling Guidance
* Outlier Treatment Recommendations
* Dataset Improvement Roadmap

### Advanced Analytics

* Correlation Analysis
* Top Feature Relationships
* Executive Summary Generation
* Dataset Intelligence Dashboard

### ML Algorithm Recommendation Engine

For Regression Datasets:

* Linear Regression
* Random Forest Regressor
* XGBoost Regressor

For Classification Datasets:

* Logistic Regression
* Random Forest Classifier
* XGBoost Classifier
